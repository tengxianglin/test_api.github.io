Apply a custom channel in the Kraus representation on the input state.

  • list_kraus_oper (Iterable[Tensor]) – Kraus operators of this channel.

avocado.operator.channel.functional.common.stinespring_repr(state, stinespring_mat, qubit_idx, dtype, backend)

Apply a custom channel in the Stinespring representation on the input state. stinespring_mat is a \((d_1 * d_2) \times d_1\) rectangular matrix. Here \(d_1\) is the dimension of space where qubit_idx locates, \(d_2\) is the dimension of auxillary system. With Dirac notation stinespring_mat can be defined as

\[\text{stinespring_mat.reshape}([d_1, d_2, d_1])[i, j, k] = \langle i, j| A |k \rangle\]

with \(A\) being the Stinespring operator and the channel acting as \(\rho \mapsto \text{tr}_2 (A \rho A^\dagger)\).

Parameters:

  • state (State) – Input quantum state

  • stinespring_mat (Tensor) – Stinespring representation \(A\) for the channel

  • qubit_idx (List[int] | int) – Index of the qubit on which the channel acts on

  • dtype (str) – Type of data

  • backend (Backend) – Backend on which the simulation is run

Returns:

Output state.

Return type:

State
